# Leave Management System (Django)

A simple and robust **Employee Leave Management System** built with **Django**, allowing employees to apply for leave, managers to approve/reject leave, and HR/Admin to manage overall leave data.  
This project helps streamline leave requests and provides a clean user interface for all user roles.
